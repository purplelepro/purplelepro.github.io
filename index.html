== Features & Commands ==

• Admin Commands

- !donator - Gives users donator commands and settings
!donator Name / ID
- !cfg - downloads a users cfg to the server allowing users to execute it
!cfg Name / ID
- !dumpuser - !finger alternative with more information
!dumpuser Name / ID
- !scan - scans a players config looking for known cheat cvars
!scan Name / ID

• User Commands

• VGuard

- Level IP White listing
- Cvar Scanner
- anti rcon / ref usage
- Player logging
- Admin notifications on "malicious" events
- DRi key - Unique server sided key (guid) for DRi* Admins level 2+
set vg_drikey "key"
